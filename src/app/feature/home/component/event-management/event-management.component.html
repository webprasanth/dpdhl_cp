Event Management

flow  

Users will be notified over Email, SMS or any other configured medium from DHL IoT Platform on qualifying certain threshold mapped in the Platform using Custom Templates. These threshold can be configured and saved by user as per their need and later been imposed on data received.

Platform can be integrated with any CRM System and create CASE as required.

Notification Methodology
SMS, EMAIL, CRM SMS Plugin, Email Plugin, CRM Plugin
